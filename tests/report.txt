Добавление пользователя
Присутствует возможность добавить пользователя с пустым полем имени и/или пароля

Изменение пользователя
Присутствует возможность при изменении пользователя оставить пустым поле имени и/или пароля

Добавление записей
Присутствует возможность добавления записи с использованием несуществующего id пользователя

Изменение записей
Присутствует возможность изменять записи, не принадлежащие пользователю

Время жизни токена
Время действия токена никак не ограничено
