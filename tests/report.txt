Добавление пользователя
Присутствует возможность добавить пользователя с пустым полем имени и/или пароля (test_add_user_negative)

Изменение пользователя
Присутствует возможность при изменении пользователя оставить пустым поле имени и/или пароля (test_change_user_negative)

Добавление записей
Присутствует возможность добавления записи с использованием несуществующего id пользователя (test_add_record_negative)

Изменение записей
Присутствует возможность изменять записи, не принадлежащие пользователю (test_change_record_negative)
Метод не возвращает значение при попытке изменить id владельца записи (test_change_record_negative)

Время жизни токена
Время действия токена никак не ограничено (test_token_lifetime)
